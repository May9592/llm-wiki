# 知识库使用指南

## 自然语言触发指令

### 编译类

| 指令                 | 功能                       |
| ------------------ | ------------------------ |
| `执行编译`             | 编译所有新文章和讨论               |
| `编译新文章`            | 处理 raw/articles/ 中的新内容   |
| `处理 raw/articles/` | 同上                       |
| `编译论文`             | 处理 raw/papers/ 中的论文      |
| `编译讨论`             | 处理 raw/discussions/ 中的讨论 |

### 问题处理类

| 指令 | 功能 |
|------|------|
| `处理这个文件：[路径]` | 处理指定的讨论文件 |
| `处理 raw/discussions/` | 处理所有讨论 |
| `分析这段讨论：[内容]` | 直接粘贴讨论内容分析 |
| `研究 P-001` | 研究特定问题的解决方案 |
| `为 P-002 收集资料` | 为问题收集相关资料 |
| `为 P-xxx 生成方案` | 生成解决方案 |

### 查询类

| 指令 | 示例 |
|------|------|
| 查询知识 | `关于 Seedance 我的知识库有什么？` |
| 查询工具 | `AI 视频制作有哪些工具？` |
| 查询问题 | `有哪些未解决的问题？` |

### 维护类

| 指令 | 功能 |
|------|------|
| `运行健康检查` | 检查知识库健康状况 |
| `检查链接` | 验证所有链接是否正确 |

## 使用场景

### 场景 1：保存新文章
```
帮我保存这篇文章：[URL]
```

### 场景 2：处理讨论
```
处理这个文件：/Users/xxx/Telegram/messages.html
```

### 场景 3：解决问题
```
1. 处理讨论 → 自动提取问题
2. 研究 P-001 → 收集资料
3. 为 P-001 生成方案 → 创建解决方案
```

### 场景 4：查询知识
```
关于 AI 视频制作我有什么资料？
```

## Skill 自动触发

以下关键词会自动触发对应 Skill：

| Skill | 触发词 |
|-------|--------|
| 讨论处理器 | `处理讨论`、`分析聊天记录`、`提取问题` |
| 编译器 | `执行编译`、`编译新文章` |
| Seedance 提示词 | `Seedance`、`视频提示词` |
