# 知识库使用指南

## 自然语言触发指令

### 编译类

| 指令 | 功能 |
| ------------------ | ------------------------ |
| `执行编译` | 编译所有新文章 |
| `编译新文章` | 处理 `raw/articles/` 中的新内容 |
| `处理 raw/articles/` | 同上 |

### 查询类

| 指令 | 示例 |
|------|------|
| 查询知识 | `关于 Graphify 我的知识库有什么？` |
| 查询工具 | `知识图谱工具有哪些？` |
| 查询概念 | `什么是社区发现？` |

### 代码分析类（Graphify）

| 指令 | 功能 |
|------|------|
| `/graphify raw/code` | 生成代码知识图谱 |
| `/graphify query "..."` | 查询图谱 |
| `/graphify path "A" "B"` | 路径查找 |
| `/graphify explain "X"` | 节点解释 |

### 维护类

| 指令 | 功能 |
|------|------|
| `运行健康检查` | 检查知识库健康状况 |
| `检查链接` | 验证所有链接是否正确 |

## 主要功能

### 1. 知识库编译
- **逐篇摘要**：生成文档摘要到 `wiki/summaries/`
- **概念抽取**：提取概念到 `wiki/concepts/`
- **索引维护**：更新 `wiki/indexes/`
- **健康检查**：每周检查知识库质量

### 2. 代码知识图谱（Graphify）
- **AST 分析**：自动提取代码结构
- **设计意图**：从注释提取 rationale
- **社区发现**：自动识别功能模块
- **可视化**：生成交互式图谱

### 3. 智能查询
- **知识查询**：搜索已编译的知识
- **图谱查询**：查询代码关系和结构
- **跨文档关联**：发现概念之间的联系

## 下一步建议

1. **编译新文章**：说 `执行编译`
2. **探索代码**：使用 `/graphify raw/code`
3. **查询知识**：直接提问或使用索引
4. **定期维护**：每周运行健康检查
