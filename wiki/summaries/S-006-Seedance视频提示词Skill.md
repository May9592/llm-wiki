# S-006: Seedance Prompt Skill - AI 视频提示词生成

## TL;DR
Claude Code Skill，将 Claude 变成字节跳动 Seedance 2.0（即梦）平台的专业 AI 视频提示词工程师。

## 核心结论

### 十大核心能力
1. 纯文本生成视频
2. 一致性控制（角色/产品/场景）
3. 镜头与动作复刻
4. 创意模板/VFX 复刻
5. 故事补全
6. 视频延长
7. 声音控制
8. 一镜到底
9. 视频编辑
10. 音乐节拍同步

### 高级技巧
- **时间戳分镜**：15秒视频精确到每秒控制（`0-3s: ...`, `4-8s: ...`）
- **技术参数指定**：长宽比、帧率、调色、镜头风格
- **负向提示词**：明确排除不需要的元素
- **多段拼接**：超过15秒自动拆分为延长链接的段落

### 多模态参考系统
- 图片 `@图片1` ~ `@图片9`
- 视频 `@视频1` ~ `@视频3`
- 音频 `@音频1` ~ `@音频3`

## 关键证据

### Seedance 2.0 平台规格
| 维度 | 规格 |
|------|------|
| 图片输入 | jpeg/png/webp/bmp/tiff/gif，最多9张，每张<30MB |
| 视频输入 | mp4/mov，最多3个，总长2-15s，每个<50MB，480p-720p |
| 音频输入 | mp3/wav，最多3个，总长≤15s，每个<15MB |
| 生成时长 | 4-15秒 |
| 输出分辨率 | 最高2K |

### 场景特定提示策略
- 电商/广告：产品展示、360度旋转、3D爆炸图
- 短剧/对话：脚本场景、角色对话、情绪标签、音效设计
- 仙侠/奇幻动画：电影级武侠、法术效果、变身序列
- 科普教育：4K医学CGI、透明解剖可视化
- 音乐MV/节拍同步：宽屏构图、多图片节拍匹配
