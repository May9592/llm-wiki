# S-014: Agency Agents - AI 代理集合

## TL;DR
一个包含 144+ 专业 AI 代理的开源集合，覆盖 12 个部门，每个代理都有明确的个性、工作流程和交付标准，支持多工具集成（Claude Code、GitHub Copilot、Cursor、Aider 等）。

## 核心结论

### 项目定位
- **不是通用提示词库**：每个代理都有深度专业化和个性
- **不是黑盒工具**：透明、可 fork、可适应的代理个性
- **完整代理系统**：包含工作流程和交付标准

### 12 大部门
1. **Engineering**（工程）：22 个代理（前端、后端、移动端、AI、DevOps 等）
2. **Design**（设计）：8 个代理（UI、UX、品牌、视觉等）
3. **Paid Media**（付费媒体）：7 个代理（PPC、SEO、追踪、创意等）
4. **Sales**（销售）：8 个代理（外拓、发现、成单、提案等）
5. **Marketing**（营销）：26 个代理（增长、内容、社交媒体、中国平台等）
6. **Product**（产品）：5 个代理（优先级、趋势、反馈、PM 等）
7. **Project Management**（项目管理）：6 个代理
8. **Testing**（测试）：8 个代理（QA、性能、API、无障碍等）
9. **Support**（支持）：6 个代理（客服、分析、财务、基础设施等）
10. **Spatial Computing**（空间计算）：5 个代理
11. **Game Development**（游戏开发）：15 个代理（跨引擎 + Unity/Unreal/Godot/Blender/Roblox）
12. **Academic**（学术）：5 个代理（人类学、地理、历史、心理学等）

### 多工具集成支持
- Claude Code（原生）
- GitHub Copilot（原生）
- Cursor（.mdc 规则）
- Aider（CONVENTIONS.md）
- Windsurf（.windsurfrules）
- OpenClaw、Gemini CLI、Antigravity、Qwen Code、Kimi Code 等

### 代理设计哲学
每个代理包含：
- 🎭 **强个性**：真实角色和声音
- 📋 **清晰交付物**：具体输出
- ✅ **成功指标**：可衡量结果
- 🔄 **经过验证的工作流程**
- 💡 **学习记忆**

## 关键证据

1. **初创 MVP**：前端 + 后端 + 增长 + 原型 + 质检

3. **企业功能开发**：项目经理 + 高级开发 + UI + 实验 + 证据收集
4. **付费媒体接管**：审计 + 追踪 + PPC + 搜索词 + 创意
