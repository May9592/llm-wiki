# MCP (Model Context Protocol)

## 定义
模型上下文协议。一种让 AI agent 连接外部工具的协议标准。

## 传输方式
- stdio（标准输入输出）
- SSE（Server-Sent Events）
- HTTP

## 在 Claudian 中的应用
- 连接外部工具和服务
- Claude 管理 vault MCP
- Codex 使用 CLI 管理的 MCP 配置
