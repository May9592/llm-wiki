# 智能素材路由

## 定义
根据 URL 域名自动选择最佳内容提取方式的机制。

## 路由规则
| URL 类型 | 提取工具 |
|----------|----------|
| 网页文章 | baoyu-url-to-markdown |
| X/Twitter | baoyu-url-to-markdown |
| 微信公众号 | wechat-article-to-markdown |
| YouTube | youtube-transcript |
| 知乎 | baoyu-url-to-markdown |
| 小红书 | 手动粘贴 |

## 回退机制
如果自动提取失败，系统会提示改走手动入口。
