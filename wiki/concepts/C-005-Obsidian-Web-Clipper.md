# Obsidian Web Clipper

## 定义
Obsidian 的浏览器扩展，将网页文章转换为 Markdown 格式并保存到 Vault 中。

## 核心功能
- 一键保存网页为 Markdown
- 自动提取标题、正文、图片
- 保留原文链接
- 可配置模板和元数据

## 配置建议
- 设置附件存储路径
- 绑定下载快捷键 Ctrl+Shift+D
- 模板中强制写入元数据（source_url、author、published、tags）
