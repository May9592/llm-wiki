# Agentic Workflow（代理工作流）

## 定义
AI Agent 执行任务的标准化、可复用流程。包含任务分解、执行步骤、质量检查和交付确认等环节。

## 核心要素
1. **任务理解**：明确需求和边界条件
2. **执行步骤**：分阶段的工作流程
3. **质量检查**：自我验证和标准符合性检查
4. **交付确认**：输出成果的格式化和交付

## 工作流示例
**Frontend Developer Agent 工作流**：
1. 分析需求 → 2. 设计组件结构 → 3. 实现代码 → 4. 性能检查 → 5. 交付文档

**Reality Checker Agent 工作流**：
1. 证据收集 → 2. 问题发现（默认 3-5 个）→ 3. 视觉证明要求 → 4. 生产就绪认证

## 价值
- **一致性**：每次执行都遵循相同标准
- **可预测性**：输出结果可预期
- **可审计性**：步骤透明可追溯
