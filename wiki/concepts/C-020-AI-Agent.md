# AI Agent（AI 代理）

## 定义
具有明确角色、个性、工作流程和交付标准的专业化 AI 实体。不同于通用聊天机器人，AI Agent 是为特定领域深度定制、有明确职责边界的专家实体。

## 核心特征
1. **🎭 强个性**：独特的角色声音、沟通方式和行事风格
2. **📋 清晰交付物**：具体、可验证的输出成果
3. **✅ 成功指标**：可衡量的质量标准和完成标准
4. **🔄 经过验证的工作流程**：步骤化的操作流程
5. **💡 学习记忆**：模式识别和持续改进能力

## 与通用提示词的区别
| 特征 | 通用提示词 | AI Agent |
|------|-----------|----------|
| 定位 | 通用角色 | 深度专业化 |
| 个性 | 模板化 | 独特角色 |
| 输出 | 模糊指导 | 具体交付物 |
| 流程 | 一次性 | 可复用工作流 |

## 设计哲学
> "每个代理都是一个专业专家，拥有个性、流程和经过验证的交付成果。"
