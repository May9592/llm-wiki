# Inline Edit (内联编辑)

## 定义
直接在笔记中编辑文本的功能，带词级 diff 预览。

## 使用方式
- 选中文本 + 热键
- 在光标位置启动

## 特点
- 词级差异预览
- 直接在笔记中编辑
- 实时查看修改
